🔐 2. BACKEND – PROVIDER ABSTRACTION LAYER

Create a folder:

/services/integrations/

Inside, create:

BaseFinancialProvider.ts
QuickBooksProvider.ts
XeroProvider.ts
FreshBooksProvider.ts
ZohoProvider.ts
BaseFinancialProvider Interface

Define a standard interface:

export interface FinancialProvider {
  connect(authCode: string): Promise<void>;
  refreshToken(): Promise<void>;
  fetchClients(): Promise<FinancialClient[]>;
  fetchInvoices(): Promise<FinancialInvoice[]>;
  fetchPayments(): Promise<FinancialPayment[]>;
  fetchEstimates(): Promise<FinancialEstimate[]>;
  disconnect(): Promise<void>;
}

All providers must implement this interface.
